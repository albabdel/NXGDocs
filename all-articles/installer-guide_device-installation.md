# Device Installation

Comprehensive guide for device installation.

## What's New

| Feature | Description | Available Since |
| :--- | :--- | :--- |
| **Device Activation/Deactivation** | Activate or deactivate devices through configuration menu to control whether sensors are operational. | March 2026 |
| **Device Creation in Sites** | Add new devices to existing sites with automatic sensor discovery. | March 2026 |
| **Device Deletion** | Delete devices from sites, removing all associated sensors and configurations. | March 2026 |
| **Device Rediscovery** | Rediscover previously deleted devices to recover configurations without manual reconfiguration. | March 2026 |
| **Sensor Activation/Deactivation** | Granular control over individual sensors within devices. | March 2026 |
| **Device Group Creation** | Create device groups to organize and manage devices logically. | March 2026 |

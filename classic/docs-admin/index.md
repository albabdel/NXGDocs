# Company Admin Documentation

Welcome to the **Company Admin** documentation for NXGEN GCXONE.

As a Company Admin, you have full access to system configuration, user management, and advanced features.

## Key Responsibilities

### System Configuration
- Configure system-wide settings and preferences
- Manage integrations and API connections
- Set up automation rules and workflows

### User Management
- Create and manage user accounts
- Assign roles and permissions
- Monitor user activity and access logs

### Device Management
- Add and configure devices
- Monitor device health and performance
- Manage device groups and hierarchies

### Reporting
- Generate system-wide reports
- Create custom dashboards
- Export data for analysis

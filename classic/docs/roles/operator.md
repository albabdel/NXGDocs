# Operator Guide

As an **Operator**, your primary responsibility is to monitor the system for alarms and take appropriate action. The GCXONE Dashboard is your command center.

## The Dashboard

The Dashboard provides a real-time overview of all connected sites and devices.
*   **Active Alarms**: High-priority events requiring immediate attention.
*   **Device Status**: Health check of all gateways and sensors.
*   **Recent Activity**: A live feed of system events.

## Handling Alarms

When an alarm is triggered (e.g., "Motion Detected" or "Door Forced Open"):

1.  **Notification**: You will hear an audible alert and see a pop-up on the dashboard.
2.  **Acknowledge**: Click the alarm to **Acknowledge** it. This lets other operators know you are handling it.
3.  **Investigate**: Review the event details, including timestamp, location, and any associated video footage.
4.  **Resolve**: Once the issue is handled, click **Close Alarm** and select a resolution code (e.g., "False Alarm", "Police Dispatched").

## Reports

You can generate reports to analyze system activity over time.
1.  Navigate to the **Reports** tab.
2.  Select a Report Type (e.g., "Alarm Summary", "User Access").
3.  Choose a Date Range.
4.  Click **Generate PDF** or **Export CSV**.

# 🕒 NTP Server Configuration

Accurate time synchronization is critical for security. It ensures that event logs, video playback, and alarm triggers are perfectly aligned across your devices and the cloud.

<Callout type="warning" title="Why Time Matters">
If your device time is out of sync, your video evidence may be inadmissible, and alarms might appear in the wrong order in the Talos CMS.
</Callout>

---

## 🌐 GCXONE NTP Servers

Use the following addresses based on your network setup:

| Setup Type | NTP Server Address | Port |
| :--- | :--- | :--- |
| **VPN / Private Network** | `time1.nxgen.cloud` | 123 (UDP) |
| **Public Internet** | `timeext1.nxgen.cloud` | 123 (UDP) |

---

## 🏗️ Step-by-Step Configuration

<Steps>

### 1. Enable TimeSync in Marketplace
Before configuring your hardware, ensure the TimeSync service is active.
1. Navigate to the **Marketplace** in GCXONE.
2. Locate the **TimeSync** module and click **Activate**.
3. Complete the short activation wizard.

### 2. Configure Device Time Zone
1. Go to the **Configurations** → **Devices** tab.
2. Click **More** on your target device and select **Time Zone Sync**.
3. Select the correct local time zone and ensure **Daylight Saving Time (DST)** is toggled **ON**.

### 3. Apply NTP to Hardware
Log in to your camera or NVR's local web interface to apply the settings.
1. Find the **Network** or **System** settings menu.
2. Enable the **NTP Client**.
3. Enter `time1.nxgen.cloud` (or `timeext1.nxgen.cloud`) as the Server Address.
4. Set the **Sync Interval** to `3600` seconds (1 hour).

</Steps>

## 📱 Manufacturer Specifics

### Hikvision / Dahua
- **Port:** 123 UDP.
- **Tip:** Stop recording temporarily before making manual time changes to prevent timeline corruption.

### Teltonika Routers
- **DNS Rebind Protection:** Must be **Disabled** in the router's DNS settings to allow the resolution of the `nxgen.cloud` domains.
- **Interval:** Set to 60 minutes for optimal balance between accuracy and network load.

---

## 🛠️ Troubleshooting

- **Sync Failed:** Verify your firewall allows outbound traffic on **Port 123 (UDP)**.
- **Off by 1 Hour:** Ensure **Daylight Saving Time** is enabled on BOTH the hardware and the GCXONE dashboard.
- **Manual Drift:** If the clock drifts significantly, reduce the sync interval to `1800` seconds.

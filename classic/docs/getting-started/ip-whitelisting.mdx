# 🛡️ IP Whitelisting Reference

To ensure seamless connectivity and secure communication between your devices and the GCXONE platform, it is essential to permit traffic to the following IP addresses on your network's firewall.

## ☁️ Core Platform Gateways

These IP addresses are mandatory for the core platform functionality.

| Service | IP Address | Description |
| :--- | :--- | :--- |
| **Primary Gateway** | `18.185.17.113` | Main platform handshake |
| **Backup Gateway** | `3.124.50.242` | Global failover node |
| **Messaging Gateway** | `3.127.50.212` | Secure event delivery |

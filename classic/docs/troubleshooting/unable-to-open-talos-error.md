# Unable to Open Evalink Talos - Oops Something Went Wrong Error

This guide helps you troubleshoot and resolve the "Oops Something Went Wrong" error when attempting to open Evalink Talos.

## Quick Checklist

- [ ] Clear browser cache and cookies
- [ ] Try a different browser
- [ ] Check browser console for error messages
- [ ] Verify network connectivity
- [ ] Check if Talos service is operational
- [ ] Verify user permissions and access rights
- [ ] Try logging out and logging back in

## Common Causes

1. **Browser Cache Issues**: Stale cache data causing conflicts
2. **Browser Compatibility**: Unsupported browser or outdated version
3. **Network Connectivity**: Connection issues to Talos servers
4. **Service Outage**: Temporary Talos service interruption
5. **Permission Issues**: Insufficient user permissions
6. **Session Expired**: User session has expired

## Troubleshooting Steps

### Step 1: Clear Browser Cache and Cookies

1. Open your browser's developer tools (F12)
2. Right-click on the refresh button
3. Select "Empty Cache and Hard Reload"
4. Alternatively, clear all browser data:
   - Chrome/Edge: Settings → Privacy → Clear browsing data
   - Firefox: Settings → Privacy → Clear Data
   - Safari: Preferences → Privacy → Manage Website Data

### Step 2: Try a Different Browser

Test if the issue is browser-specific:

- Try Chrome, Firefox, Edge, or Safari
- Ensure the browser is up to date
- Disable browser extensions temporarily

### Step 3: Check Browser Console

1. Open browser developer tools (F12)
2. Navigate to the Console tab
3. Look for error messages
4. Note any specific error codes or messages
5. Share these with support if the issue persists

### Step 4: Verify Network Connectivity

1. Check your internet connection
2. Verify you can access other websites
3. Test connection to Talos servers
4. Check firewall settings if on a corporate network

### Step 5: Check Service Status

1. Verify if Talos service is operational
2. Check [GCXONE System Status](/docs/support/system-status) if available
3. Contact support to verify service availability

### Step 6: Verify User Permissions

1. Ensure your user account has proper permissions
2. Verify your role has access to Talos
3. Check if your account is active
4. Contact your administrator to verify access rights

### Step 7: Try Logging Out and Back In

1. Log out of your current session
2. Clear browser cache (as in Step 1)
3. Log back in with your credentials
4. Try accessing Talos again

## Advanced Troubleshooting

### Check Browser Compatibility

Talos requires a modern browser with JavaScript enabled. Supported browsers:

- Chrome (latest version)
- Firefox (latest version)
- Edge (latest version)
- Safari (latest version)

### Disable Browser Extensions

Some browser extensions can interfere with Talos:

1. Disable all extensions temporarily
2. Try accessing Talos
3. If it works, enable extensions one by one to identify the problematic extension

### Check Network Settings

If on a corporate network:

1. Verify proxy settings
2. Check firewall rules
3. Ensure Talos domains are whitelisted
4. Contact IT support if needed

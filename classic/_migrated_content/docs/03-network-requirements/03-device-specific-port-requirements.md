# Device-Specific Port Requirements

In addition to the core gateways, specialized receiver IPs are required for certain devices (which must be whitelisted):
Dahua Receiver: 52.59.60.20.
Hikvision Receiver: 35.156.60.98.
Hanwha Alarm Receiver Gateway: 18.184.110.24.
Milestone Alarm Receiver Gateway: 3.66.98.181.
ADPRO Receiver: Tenant-specific IP on Port 10000.
Camect Receiver: 3.122.169.231 (in addition to gateways).

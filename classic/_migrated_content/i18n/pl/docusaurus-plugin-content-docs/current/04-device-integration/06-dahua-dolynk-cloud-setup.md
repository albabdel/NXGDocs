# Dahua DoLynk Cloud Setup

Prerequisites (DoLynk): Installer must provide Serial Number, Device Password, User Name, and Category code. For sharing with NXGEN, the email dev@nxgen.info (NXGEN Technology AG) must be used in the Company Name/email field within the DoLynk Care account.
Supported Modes: Supports both Cloud Mode (integration with Dahua DoLynk Cloud) and Local Mode.
Local Mode: Local mode operations are supported for advanced functionalities like P2P streaming, two-way audio, and encrypted streams on operator workstations. Local Mode requires prerequisites like Microsoft Redistributables, .NET Framework 4.8, and NodeJS (V18 preferred).
Supported Features: Live Stream, Playback, Events/Alarms (with analytics), PTZ Control, I/O Management.
Time Synchronization: Resolution for incorrect time settings is to use timel.nxgen.cloud as the NTP server with an interval set appropriately (e.g., 3600 seconds).

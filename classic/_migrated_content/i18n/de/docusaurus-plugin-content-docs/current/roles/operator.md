# Operator-Leitfaden

Als **Operator** überwachen Sie primär das System auf Alarme und reagieren angemessen. Das GCXONE Dashboard ist dabei Ihr Control Center.

## Das Dashboard

Das Dashboard bietet einen Echtzeitüberblick über alle verbundenen Sites und Devices.
* **Active Alarms**: Hochpriorisierte Ereignisse, die sofortige Aufmerksamkeit benötigen.
* **Device Status**: Health-Check aller Gateways und Sensoren.
* **Recent Activity**: Live-Feed der Systemevents.

## Alarme bearbeiten

Wenn ein Alarm ausgelöst wird (z. B. „Motion Detected“ oder „Door Forced Open“):

1. **Notification**: Akustischer Alarm und Pop-up im Dashboard.
2. **Acknowledge**: Alarm anklicken und **Acknowledge** wählen, damit andere Operator sehen, dass Sie übernehmen.
3. **Investigate**: Event-Details prüfen, inklusive Zeitstempel, Standort und zugehörigem Videomaterial.
4. **Resolve**: Nach Bearbeitung **Close Alarm** wählen und einen Resolution Code setzen (z. B. „False Alarm“, „Police Dispatched“).

## Reports

Berichte helfen, die Systemaktivität über die Zeit zu analysieren.
1. Zum Tab **Reports** navigieren.
2. Einen Report Type wählen (z. B. „Alarm Summary“, „User Access“).
3. Date Range auswählen.
4. Auf **Generate PDF** oder **Export CSV** klicken.

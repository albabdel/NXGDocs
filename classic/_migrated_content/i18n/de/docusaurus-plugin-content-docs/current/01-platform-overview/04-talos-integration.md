# GCXONE-Integration mit Evalink Talos CMS

## Überblick
Evalink Talos ist für das Alarmmanagement entscheidend. GCXONE verknüpft sich über API-Tokens/-Schlüssel auf Root-Konfigurationsebene mit Talos, damit Alarme nahtlos synchronisiert werden.

## Schlüsselkonzepte
- **Rolle von Talos**: Zentrales Managementsystem (CMS) zur Alarmbearbeitung.
- **Root-Konfiguration**: Integration wird auf oberster Ebene eingerichtet.
- **API-Tokens**: Sichere Verbindung über API-Schlüssel.

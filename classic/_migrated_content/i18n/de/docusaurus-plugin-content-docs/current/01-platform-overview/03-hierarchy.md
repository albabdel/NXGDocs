# Die GCXONE-Hierarchie verstehen (Die 5-Ebenen-Struktur)

## Überblick
Ein klares Verständnis der hierarchischen Organisation von GCXONE ist entscheidend für ein wirksames Management. Die Plattform nutzt eine 5-Ebenen-Struktur, um Assets und Berechtigungen zu gliedern.

## Die 5-Ebenen-Struktur
1. **Tenant**: Das übergeordnete Dienstleisterkonto.
2. **Customer**: Die Endkundenorganisation.
3. **Site**: Ein physischer Standort eines Kunden.
4. **Device**: Hardwareeinheiten (NVRs, Kameras, IoT-Gateways).
5. **Kameras/Sensoren**: Einzelne Kanäle oder Sensoren, die an ein Device angeschlossen sind.

# GCXONE: Vereinheitlichtes Sicherheitsmanagement (SaaS)

## Überblick
GCXONE ist eine cloudbasierte Software-as-a-Service-(SaaS)-Plattform, die Videoüberwachungs- und IoT-Dienste bereitstellt. Sie fungiert als videofokussierter Unified Security Management Service (USMS).

## Schlüsselkonzepte
- **SaaS-Modell**: Cloudbasierte Bereitstellung, keine lokale Serverwartung erforderlich.
- **USMS**: Vereint Video-, IoT- und Alarmmanagement in einer einzigen Plattform.
- **Videozentriert**: Primärer Fokus auf Videoverifizierung und Überwachung.

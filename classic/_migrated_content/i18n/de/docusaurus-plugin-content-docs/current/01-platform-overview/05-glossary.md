# Glossar der GCXONE- und Sicherheitsbegriffe

## Definitionen
- **NVR**: Network Video Recorder (Netzwerkvideorekorder).
- **RTSP**: Real-Time Streaming Protocol.
- **SIP**: Session Initiation Protocol (für Audio).
- **Webhook**: Verfahren, mit dem eine Anwendung Echtzeitdaten an eine andere sendet.
- **PTZ**: Pan-Tilt-Zoom-Kamerasteuerung.
- **I/O**: Input/Output für Sensoren und Relais.
- **Analytics**: Automatisierte Videoanalyse (z. B. Bewegungserkennung, Linienüberquerung).
- **Alarmfilterung**: Logik zur Reduzierung von Fehlalarmen.

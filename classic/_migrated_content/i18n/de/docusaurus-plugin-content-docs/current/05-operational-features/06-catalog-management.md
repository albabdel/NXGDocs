# Geräte- und Sensormanagement über den Catalog

## Überblick
Der Catalog dient als zentrale Ablage zur Verwaltung aller Hardware-Assets.

## Assets
- **Devices**: NVRs, Kameras.
- **Gateways**: Konnektivitäts-Hardware.
- **Sensors**: IoT-Endpunkte.

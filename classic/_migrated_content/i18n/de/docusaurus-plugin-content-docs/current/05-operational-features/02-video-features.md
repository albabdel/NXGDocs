# Livevideo, Playback und Timeline-Funktionen

## Überblick
So greifen Sie auf Echtzeitstreams zu und prüfen archiviertes Videomaterial.

## Funktionen
- **Live Video**: Monitoring in Echtzeit.
- **Playback**: Historisches Videomaterial abspielen.
- **Timeline**: Chronologische Ansicht zur Ereignisrekonstruktion.

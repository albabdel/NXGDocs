# Maps, Zonen und Geofencing verwenden

## Überblick
Kamerastandorte visualisieren und geografische Grenzen definieren.

## Funktionen
- **Map View**: Assets auf einer Karte anzeigen.
- **Geo-fencing**: Aktionen basierend auf Standort auslösen.
- **Zones**: Bereiche von Interesse definieren.

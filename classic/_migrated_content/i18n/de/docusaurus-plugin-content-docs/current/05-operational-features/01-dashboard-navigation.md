# Navigation im GCXONE-Dashboard und den Hauptanwendungen

## Überblick
Ein Leitfaden zu den wichtigsten Anwendungen in der GCXONE-Plattform.

## Zentrale Apps
- **Dashboard**: Übersicht auf hoher Ebene.
- **Configuration**: System-Setup.
- **Talos CMS**: Alarmmanagement.
- **Video Viewer**: Live- und aufgezeichnetes Video.
- **Map**: Geografische Darstellung von Assets.

# Synchronisierung von direkt in Talos erstellten Standorten

## Überblick
Wenn ein Standort direkt in Talos angelegt wird (außerhalb des regulären Forward-Sync-Prozesses), sind manuelle Schritte nötig, um die Daten in GCXONE abzurufen.

## Vorgehensweise
1. Standort in Talos identifizieren.
2. Über die GCXONE-Oberfläche eine manuelle Synchronisierung auslösen.
3. Datenkonsistenz prüfen.

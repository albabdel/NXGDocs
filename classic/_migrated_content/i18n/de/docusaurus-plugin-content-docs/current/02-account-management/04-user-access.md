# Benutzer, Rollen und Berechtigungen verwalten

## Überblick
Diese Anleitung beschreibt das Hinzufügen interner Nutzer (Admins, Operators) und Kundenbenutzer, das Festlegen von Sitzungs-Timeouts und das Anpassen von Zugriffsrechten.

## Wichtige Aktionen
- **Benutzer hinzufügen**: Neue Benutzerkonten erstellen.
- **Rollen zuweisen**: Im Tab „Roles“ Berechtigungen definieren (z. B. Read-Only, Admin).
- **Sitzungs-Timeouts**: Sicherheitseinstellungen für Benutzersitzungen konfigurieren.

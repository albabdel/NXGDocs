# Checkliste für das Onboarding neuer Mandanten

## Überblick
Diese Checkliste beschreibt die verpflichtenden Daten für die Ersteinrichtung eines neuen Mandanten.

## Checkliste
- [ ] **Mandantenname**: Offizieller Name der Organisation.
- [ ] **Kontaktdaten**: Hauptansprechpartner.
- [ ] **Admin-/Operator-E-Mails**: Liste der Nutzer, die Zugriff benötigen.
- [ ] **Talos-Kundenstatus**: Bestätigung des Talos-Kontos.
- [ ] **Workflow-Anforderungen**: Erste Anforderungen an Workflows.
- [ ] **Geräteliste**: Inventar der Geräte, die eingebunden werden sollen.

# Gerätestatus prüfen und Konnektivität beheben

## Überblick
Konnektivitätsprobleme mit integrierten Tools erkennen und lösen.

## Tools
- **Audit Log**: Auf Fehlermeldungen prüfen.
- **Dashboard**: Notifications überwachen (z. B. Event Count von nicht konfigurierten Devices).

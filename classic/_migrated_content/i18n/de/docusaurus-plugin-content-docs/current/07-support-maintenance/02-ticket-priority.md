# Ticket-Prioritäten verstehen

## Überblick
Die Ticket-Priorität bestimmt die erwartete Reaktionszeit.

## Stufen
- **Critical**: System ausgefallen.
- **High**: Wichtige Funktion stark beeinträchtigt.
- **Medium**: Kleinere Beeinträchtigung.
- **Low**: Allgemeine Anfrage.

## Hinweis
- **Product Name**: Immer „GCXONE“ oder „GCXONE“ angeben.

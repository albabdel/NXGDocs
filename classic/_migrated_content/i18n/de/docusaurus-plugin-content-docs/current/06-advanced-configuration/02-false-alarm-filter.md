# Den False Alarm Filter aktivieren

## Überblick
Reduzieren Sie Operator-Fatigue, indem Sie den False Alarm Filter aktivieren.

## Setup
- **Enable Feature**: Filter in den Systemeinstellungen einschalten.
- **Device Config**: Sicherstellen, dass die Geräteeinstellungen Analytics unterstützen.

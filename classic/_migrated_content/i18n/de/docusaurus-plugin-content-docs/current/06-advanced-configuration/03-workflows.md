# Alarm-Workflows konfigurieren (Talos Workflows App)

## Überblick
Workflows erstellen und verwalten, um festzulegen, wie Alarme behandelt werden.

## Workflow-Typen
- **Global**: Gilt für alle Customers.
- **Managed**: Vom Service Provider verwaltet.
- **Site-Specific**: Für einzelne Sites zugeschnitten.

# Einführung in Alarmverarbeitung und Reduktion von Fehlalarmen

## Überblick
Verstehen, wie GCXONE Alarme empfängt, analysiert und klassifiziert, um Rauschen zu minimieren.

## Prozess
1. **Reception**: Alarme werden von Devices empfangen.
2. **Analysis**: Analytics bewerten die Gültigkeit.
3. **Classification**: Echtes Ereignis vs. False Alarm.
4. **Distribution**: An Operators zur Bearbeitung gesendet.

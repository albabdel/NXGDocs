# Heitel Device Integration (Manual Intervention Required)

## Überblick
Legacy-Heitel-Devices benötigen manuelle Konfigurationsunterstützung.

## Prozess
- **Ticket Required**: Help-Desk-Ticket für manuelle Unterstützung eröffnen.
- **Support**: Das NXGEN Support-Team muss den Alarmempfang manuell konfigurieren.

# Reconeyez-Integration (DC-09 an Talos, Bilder an FTP)

## Überblick
Reconeyez nutzt eine Dual-Path-Integrationsstrategie.

## Workflow
1. **Alarms**: Über SIA DC-09 direkt an Talos gesendet.
2. **Images**: Zum GCXONE FTP-Server übertragen.
3. **Forwarding**: Benötigt einen Talos-Workflow, um Bilder mit Alarmen zu verknüpfen.

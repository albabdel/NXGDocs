# Erforderliche Device-Details (Credentials, Ports, Unique IDs)

## Überblick
Für eine erfolgreiche Device-Integration sind präzise Konfigurationsdetails notwendig.

## Anforderungen
- **Privileges**: Admin-/Operator-Rechte sind zwingend erforderlich.
- **Unique IDs**: Server Unit ID und Serial Number müssen eindeutig sein, um Alarmkonflikte zu vermeiden.

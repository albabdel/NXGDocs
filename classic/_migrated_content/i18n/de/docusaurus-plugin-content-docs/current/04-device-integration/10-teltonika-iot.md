# Teltonika-IOT Integration und Custom Alarm Rules

## Überblick
Die Integration von Teltonika IoT-Devices benötigt eine angepasste Konfiguration zur Steuerung des Datenflusses.

## Konfiguration
- **Credentials**: Serial Number, Username, Password.
- **Custom Alarm Rules**: Über das Additional Property-Feld können übermäßige Event-Daten (z. B. Spannungswerte) gefiltert werden.

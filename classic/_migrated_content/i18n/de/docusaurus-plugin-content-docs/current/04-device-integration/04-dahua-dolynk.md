# Dahua DoLynk Cloud Integration (Serial No., Category Code)

## Überblick
Dahua-Devices über die DoLynk-Cloud-Plattform integrieren.

## Voraussetzungen
- **Serial Number**: Eindeutiger Geräte-Identifikator.
- **Category Code**: Device-Typ (NVR, XVR usw.).
- **Credentials**: Device User Name und Password.

# Webhook-Integrationsleitfaden: Axis IP Camera und Ganz AI BOX

## Überblick
Geräte so konfigurieren, dass sie Events an den GCXONE-Webhook-Endpunkt senden.

## Konfiguration
- **Endpoint**: `https://[proxy].nxgen.cloud/eventIngest/<nxgenDeviceId>`
- **Authentication**: `nxgenDeviceId` als Username und Password verwenden.

# Kontoeinrichtung

Befolgen Sie diese Schritte, um Ihr GCXONE-Konto einzurichten.

## Erstellen eines Kontos

1.  Navigieren Sie zur GCXONE-Anmeldeseite.
2.  Klicken Sie auf **Registrieren**.
3.  Füllen Sie Ihre Organisationsdetails und die Kontaktinformationen des Administrators aus.
4.  Bestätigen Sie Ihre E-Mail-Adresse.

## Erstkonfiguration

Sobald Sie angemeldet sind, werden Sie durch den Einrichtungsassistenten geführt, um:
- Ihren ersten Standort hinzuzufügen.
- Geräte zu verbinden.
- Teammitglieder einzuladen.

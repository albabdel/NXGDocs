# Netzwerkbereitschaft und Konnektivitätsplanung

## Überblick
Vor der Bereitstellung von GCXONE sollte die Netzwerkbereitschaft bewertet werden. Dieser Leitfaden unterscheidet Anforderungen für VPN-Kunden und Nicht-VPN-Kunden.

## Konnektivitätstypen
- **VPN-Kunden**: Nutzen interne IPs für eine gesicherte Kommunikation.
- **Nicht-VPN-Kunden**: Arbeiten mit externen IPs und Portweiterleitung.

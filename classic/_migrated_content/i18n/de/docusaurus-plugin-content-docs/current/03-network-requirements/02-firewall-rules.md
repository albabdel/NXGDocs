# Erforderliche Ports, Protokolle und Domains für Whitelisting

## Überblick
Damit die GCXONE-Webanwendung ohne Unterbrechung erreichbar ist, müssen bestimmte Domains und Ports auf die Whitelist gesetzt werden.

## Obligatorische Whitelist
- **Domains**: `*.nxgen.cloud`, Auth0, Googleapis.
- **Ports**:
    - HTTP: **80**
    - HTTPS: **443**

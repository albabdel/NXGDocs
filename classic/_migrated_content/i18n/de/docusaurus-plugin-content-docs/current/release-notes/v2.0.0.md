# Version 2.0.0

*Veröffentlicht: 30. Oktober 2023*

## 🚀 Major Release – GCXONE GCX-ONE Platform

Dieses Major Release bringt tiefgreifende Architekturverbesserungen und neue Funktionen, die das Security Operations Management neu definieren.

## ✨ Neue Funktionen

### Zen Mode
- **Ablenkungsfreie Oberfläche** – Fokus auf kritische Alerts ohne UI-Überladung.
- **Intelligente Benachrichtigungsfilter** – Nur sehen, was sofortige Aufmerksamkeit erfordert.
- **Anpassbare Fokusprofile** – Verschiedene Modi für unterschiedliche Szenarien speichern.

### Enhanced Bulk Import
- **Support für 10.000+ Sites** – Import in großem Maßstab.
- **Erweiterte Validierung** – Vorabprüfung mit detaillierten Fehlermeldungen.
- **Fortschrittsanzeige** – Echtzeitstatus des Imports mit ETA-Berechnung.

### HealthCheck Monitoring
- **Proaktives System-Monitoring** – Probleme erkennen, bevor sie Auswirkungen haben.
- **Automatisierte Diagnosen** – Self-Healing-Fähigkeiten für gängige Probleme.
- **Umfassendes Reporting** – Detaillierte Health-Metriken und Trends.

## 🔧 Wesentliche Verbesserungen

### Nova99x AI Engine
- **Kompletter Algorithmus-Refresh** – 70 % genauere Threat Detection.
- **Kürzere Verarbeitungszeiten** – 3x schnellere Analyse der Videostreams.
- **Erweiterte Lernfähigkeit** – Kontinuierliche Verbesserung durch Nutzerfeedback.

### User Interface Overhaul
- **Modernes Designsystem** – Konsistente, barrierearme und responsive Oberfläche.
- **Verbesserte Navigation** – Intuitive Menüstruktur und bessere Suche.
- **Optimierte Mobile-Unterstützung** – Vollwertige mobile Web-App.

### Performance & Scalability
- **Neue Microservices-Architektur** – Mehr Zuverlässigkeit und einfachere Wartung.
- **Horizontale Skalierung** – Für Deployments im Enterprise-Maßstab.
- **Verbessertes Caching** – Deutlich verkürzte Antwortzeiten.

## ⚠️ Breaking Changes

**Wichtig:** Dieses Release enthält Breaking Changes. Bitte den Migrationsleitfaden prüfen.

- **API v1 veraltet** – Migration zu API v2 (Support für v1 endet im März 2024).
- **Konfigurationsformat aktualisiert** – Automatisches Migrationstool vorhanden.
- **Datenbankschema geändert** – Backup vor dem Upgrade erforderlich.

## 🐞 Bug Fixes

- Über 150 gemeldete Issues aus vorherigen Versionen behoben.
- Alle bekannten Memory Leaks gelöst.
- Cross-Browser-Kompatibilitätsprobleme behoben.
- Zeitzonenhandling über verschiedene Regionen korrigiert.

## 📝 Dokumentations-Updates

- **Kompletter API-Dokumentations-Refresh** – Interaktive Beispiele und bessere Struktur.
- **Neue Deployment-Guides** – Schritt-für-Schritt-Anleitungen für verschiedene Umgebungen.
- **Video-Tutorials** – Visuelle Guides für gängige Workflows.
- **Migrationsdokumentation** – Detaillierte Upgrade-Anweisungen.

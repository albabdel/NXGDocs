# Version 2.0.5

*Veröffentlicht: 8. Dezember 2023*

## 🐞 Bug Fixes

### Kritische Fixes
- **Memory Leak** in PulseView bei langen Wiedergabesitzungen behoben.
- **Authentifizierungs-Timeouts** für Nutzer mit langen Sessions gelöst.
- **Bulk Import**-Fehler bei CSV-Dateien mit Sonderzeichen korrigiert.

### Security Patches
- **Verbesserte Input-Validierung** zur Vermeidung potenzieller XSS-Schwachstellen.
- **Aktualisierte SSL-Zertifikatsbehandlung** für höhere Sicherheit.
- **Fix für Session Management**-Race Condition bei gleichzeitigen Nutzern.

## 🔧 Kleinere Verbesserungen

- Fehlermeldungen bei fehlgeschlagenen Kameraverbindungen verbessert.
- Logging für Troubleshooting von Deployments erweitert.
- Datenbankabfragen für HealthCheck-Reports optimiert.
- UI-Übersetzungen für bessere Lokalisierung aktualisiert.

## ⚡ Performance-Optimierungen

- Speichernutzung bei Lastspitzen um 15 % reduziert.
- Videocodec-Handling für bessere Streaming-Performance verbessert.
- CustomView-Filterverarbeitung für große Datensätze optimiert.

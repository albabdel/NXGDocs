# Version 2.1.0

*Veröffentlicht: 15. Januar 2024*

## ✨ Neue Funktionen

### Nova99x Enhanced AI Engine
- **Verbesserte Erkennung von False Positives** – 45 % weniger Fehlalarme durch erweiterte Machine-Learning-Algorithmen.
- **Echtzeit-Threat-Scoring** – Dynamische Risikobewertung für präzisere Incident-Priorisierung.
- **Custom Detection Zones** – Spezifische Bereiche für präzisere Überwachung definieren.

### Marketplace Expansion
- **Neue Third-Party-Integrationen** – Support für 15+ Security-Anbieter ergänzt.
- **Custom App Deployment** – Eigene Anwendungen direkt aus dem Marketplace bereitstellen.
- **Revenue-Sharing-Programm** – Neue Monetarisierungsoptionen für Solution Partner.

### TowerGuard Automation
- **Automatisierte Deployment-Workflows** – Tower-Setup-Zeit um 60 % reduzieren.
- **Remote Configuration Management** – Mehrere Sites gleichzeitig konfigurieren.
- **Predictive Maintenance Alerts** – Proaktive Hardwareüberwachung und Alarme.

## 🔧 Verbesserungen

### Performance-Optimierungen
- **Schnelleres Videoladen** – 40 % schnellerer Start von Videostreams.
- **Optimiertes Dashboard-Rendering** – Seitenladezeiten um 25 % reduziert.
- **Bessere Mobile-Responsiveness** – Verbesserte Nutzung auf Tablets und Smartphones.

### User Experience
- **Überarbeitete Navigation** – Intuitivere Menüstruktur und bessere Suche.
- **Dark-Mode-Support** – Neues Theme für weniger Augenbelastung bei Nachtschichten.
- **Keyboard Shortcuts** – Hotkeys für häufige Aktionen und schnellere Navigation.

## 🐞 Bug Fixes

- Problem mit TimeSync behoben, das bei bestimmten Kameramodellen Zeitstempelverschiebungen verursachte.
- CustomView-Filter-Persistenz über Browsersitzungen hinweg korrigiert.
- PulseView-Playback-Steuerung in Firefox repariert.
- HealthCheck-Benachrichtigungs-Verzögerungen behoben.
- Zen Mode-Toggle speichert den Status jetzt korrekt.

## 🛠️ Technische Updates

- Upgrade auf aktuelle Sicherheitsprotokolle (TLS 1.3).
- Datenbankperformance durch optimierte Indizes verbessert.
- API-Rate-Limiting und Error Handling verstärkt.
- Third-Party-Dependencies aktualisiert für höhere Stabilität.

## ⚠️ Bekannte Probleme

- Safari-Nutzer können leichte Verzögerungen bei der Generierung von Videothumbnails erleben.
- Importvorgänge für Sites mit mehr als 10.000 Kameras können länger dauern als erwartet.

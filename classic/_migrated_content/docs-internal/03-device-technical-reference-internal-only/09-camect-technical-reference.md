# Camect Technical Reference

Live Video: Provides live view via an HLS URL.
Supported Features (Cloud Mode): Discovery, Live Video, Events, ARM/DISARM, GCXONE Audio (SIP), PTZ/Presets, Event Acknowledgement.
Limitations: Playback and Timeline features are not supported by the Camect device itself in Cloud Mode. Playback is also not supported in Local Mode.
Configuration Detail: Requires a unique HLS stream for each camera to be presented to the monitoring station technical team.

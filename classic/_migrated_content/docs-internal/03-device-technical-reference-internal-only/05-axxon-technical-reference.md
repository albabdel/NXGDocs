# Axxon Technical Reference

Event Integration: Events are pulled by GCXONE via a websocket subscription; no manual configuration is required on the Axxon side.
Supported Features (Cloud Mode): Discovery, Live Video, Playback, Timeline, Events, ARM/DISARM (software arm), GCXONE Audio (SIP), PTZ/Presets.
Limitation: I/O functionalities are NOT supported for Axxon VMS.
Known Issues: Passwords might not support special characters such as @ or #. The current alarm polling mechanism is noted as not reliable.

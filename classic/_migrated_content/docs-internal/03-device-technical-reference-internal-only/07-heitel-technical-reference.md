# Heitel Technical Reference

Nature: Considered a legacy device, often requiring manual intervention for successful alarm reception.
Onboarding: Requires raising an individual ticket (per device) in the NXGEN helpdesk portal to enable alarm reception due to its legacy nature.
Prerequisite: The device must be reachable via HTConnect.
Alarm Transmission: Requires whitelisting Heitel Gateway IPs (3.123.206.197 for video and 3.124.38.48 for events) on the customer's firewall, and Port 3333 must be open.

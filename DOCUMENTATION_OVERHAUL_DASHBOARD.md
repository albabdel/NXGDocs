## 🔄 Agent Communication Protocol

### Daily Standup (Async)
Each agent posts to dashboard:
1. **Completed yesterday:** List of completed tasks
2. **Working on today:** Current focus
3. **Blockers:** Any impediments
4. **Questions:** Items needing clarification

### Status Updates
- ✅ **Completed:** Task finished and reviewed
- 🏃 **In Progress:** Currently working
- ⏸️ **Blocked:** Waiting for dependency
- 🔄 **Review:** Ready for QA
- ❌ **Cancelled:** No longer needed

### Handoff Process
1. Complete task
2. Update dashboard
3. Create handoff document
4. Notify dependent agent
5. Schedule review
